# 🌌 Premium Hello World: Interactive Particle Universe

최신 프론트엔드 기술을 활용하여 제작된 인터랙티브한 "Hello World" 웹 페이지입니다. 다크 모드 기반의 프리미엄 디자인과 물리 엔진이 적용된 파티클 시스템을 경험해 보세요.

## ✨ 주요 기능

*   **인터랙티브 파티클 텍스트**: "HELLO WORLD" 문구가 수천 개의 파티클로 구성되어 있으며, 마우스 움직임과 클릭에 물리적으로 반응합니다.
*   **실시간 데이터 대시보드 (v3.0)**:
    *   **날짜/시간**: 1초 단위로 업데이트되는 실시간 시계.
    *   **위치 정보**: 브라우저 Geolocation API를 통한 현재 위치 좌표 표시.
*   **프리미엄 디자인**:
    *   **Glassmorphism**: 투명도와 블러 효과를 활용한 세련된 UI.
    *   **Animated Background**: 캔버스를 활용한 부드러운 메쉬 그라데이션 이동 효과.
    *   **Cursor Glow**: 마우스를 따라다니는 은은한 발광 효과.

## 🛠 사용된 기술

*   **HTML5 & CSS3**: 시맨틱 마크업 및 최신 레이아웃 기법.
*   **JavaScript (Vanilla)**: 외부 라이브러리 없이 구현된 고성능 파티클 물리 엔진 및 애니메이션 루프.
*   **Canvas API**: 배경 그래픽 및 파티클 렌더링.
*   **Git**: 버전 관리 (v1.0, v2.0, v3.0).

## 🚀 시작하기

이 프로젝트는 단일 `index.html` 파일로 구성되어 있어 별도의 설치 과정이 필요하지 않습니다.

1.  이 저장소를 클론하거나 `index.html` 파일을 다운로드합니다.
2.  브라우저(Chrome, Safari 등)에서 `index.html`을 엽니다.
3.  위치 정보 표시를 위해 상단의 '위치 권한' 요청을 승인해 주세요.
